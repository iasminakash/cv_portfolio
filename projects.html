Plantswap REST API
The PlantSwap REST API is a backend service designed to power a plant exchange platform where users can list, browse, and request plant trades. Built with Spring Boot and MongoDB, this API supports core functionalities like user authentication, plant listing management, and trade request handling. It follows RESTful principles to provide a clean, scalable, and efficient architecture.

Salary tracker
The Salary Tracker is a basic Java assignment designed to reinforce foundational programming skills. The application allows users to input and manage their monthly salary records through a simple console interface. It demonstrates core Java concepts such as variables, conditionals, loops, methods, and basic object-oriented programming (OOP).
Trascation tracker
The Transaction Tracker is a Java console application designed for students to practice basic programming concepts by tracking financial transactions. The program allows users to input transactions (both income and expenses) and keeps a running total of the balance. The application calculates and displays a summary, including the total income, total expenses, and current balance.
